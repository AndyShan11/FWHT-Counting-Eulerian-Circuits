# FWHT-Counting-Eulerian-Circuits
Exact Eulerian circuit counting for undirected graphs via the Luo–Roy homological trace formula. Combines FWHT spectral decoding, Gray-code state transitions, and OpenMP parallelization. FPT by circuit rank g, complexity O(2ᵍ·m³log m). Beats backtracking at g ≥ 9; supports 128-bit and GMP arithmetic.
